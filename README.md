## Document splitting
- `none` ‚Äî skip splitting; one embedding per page row (collection `jimdo_docs`)
- `recursive` ‚Äî `RecursiveCharacterTextSplitter` with `chunk_size`/`chunk_overlap`, good default for prose (collection suffix `_recursive-{chunk_size}-{chunk_overlap}`)
- `tokens` ‚Äî `TokenTextSplitter` (`cl100k_base`) to respect token counts, useful when downstream models are token-limited (collection suffix `_tokens-{chunk_size}-{chunk_overlap}`)
- `--chunk-size` and `--chunk-overlap` apply to `recursive` and `tokens`; use `--n-documents` for quick dry-runs (adds `_test` to the collection name)
